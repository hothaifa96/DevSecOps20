The Pizza Party Problem

The Story:

You're hosting a pizza party for your friends! You've ordered several delicious pizzas, and each pizza has 8 slices. Now comes the tricky part - you need to figure out how many slices each friend gets and if there will be any leftover slices for you to enjoy later.

Your friends are already at the door, and they're getting hungry. You need to solve this quickly before they start complaining!

üéØ Your Mission:

Create a Python program that asks how many pizzas you ordered and how many friends are coming. Then calculate:

  • How many slices each person gets
  • How many slices will be left over
  • The total number of slices available
üìö Python Concepts Used:
int() casting input() function Variables Floor division (//) Modulo (%) F-strings print() with formatting
üí° Helpful Hints:
  • Floor Division (//): Use // to get whole slices per person (no fractions!)
  • Modulo (%): Use % to find the remainder (leftover slices)
  • Remember: Each pizza has 8 slices, so multiply pizzas by 8
  • Casting: Use int() to convert user input from string to integer
  • Example: 19 // 3 = 6, and 19 % 3 = 1

The Corner Store Adventure

The Story:

You're working part-time at your neighborhood corner store. A customer walks in wanting to buy some snacks, but they only have a $20 bill. They want to know exactly how much they can spend and what their change will be.

The store owner trusts you to handle this transaction properly. You need to create a quick calculator to help the customer make their decision!

üéØ Your Mission:

Create a program that:

  • Asks for the customer's name (to be friendly!)
  • Gets the price of the items they want to buy
  • Calculates the change from a $20 bill
  • Determines if they have enough money

The Gaming Tournament

The Story:

You're organizing a gaming tournament at your local community center. Players need to register, and you need to check if they meet the requirements: they must be at least 13 years old and have their own gaming controller.

The tournament is starting soon, and you have a line of excited gamers waiting to sign up. You need a quick registration system!

üéØ Your Mission:

Create a registration system that:

  • Gets the player's name and age
  • Asks if they have their own controller (yes/no)

  • Shows their registration status

The Secret Message

The Story:

You and your best friend have decided to create a secret messaging system! You want to encode messages by creating special patterns and formatting. Your friend is waiting for you to send the first coded message.

The code works by taking a person's name and a secret word, then creating a special formatted message with patterns and repetitions.

üéØ Your Mission:

Create a secret message encoder that:

  • Gets the sender's name and recipient's name
  • Gets a secret word to encode
  • Creates a decorative border using string repetition
  • Formats everything into a secret message

All Data Types Variables Type Casting Mathematical Operations Conditional Statements For Loops Boolean Logic F-strings Built-in Functions String Formatting
üí° Helpful Hints:

  • Multiple conditions: Use elif for grade ranges (90+, 80+, 70+, etc.)
  • Built-in functions: max() and min() find highest/lowest values

  • Complex calculations: Break down weighted averages step by step
  • Boolean expressions: Store comparison results in variables
