1.
What does the input() function do? What type does it return?

Correct Answer:

The input() function reads a line of text from the user (keyboard input) and returns it as a string data type. It always returns a string, even if the user enters numbers.

name = input("Enter your name: ") print(type(name)) # <class 'str'>

3.
What does the split() function do? What type does it return? If you want the separation in split() to not be by space, how can you do that?

Correct Answer:

The split() function splits a string into a list of substrings. By default, it splits by whitespace (spaces).

To split by a different character, pass it as a parameter: split(",") or split(";")

text = "apple,banana,cherry" words = text.split(",") # Split by comma print(words) # ['apple', 'banana', 'cherry'] sentence = "Hello World Python" words2 = sentence.split() # Split by space (default) print(words2) # ['Hello', 'World', 'Python']
4.
What does the format() function do? How does it save on writing?

Correct Answer:

The format() function allows you to insert values into strings using placeholders {}. It saves writing by avoiding string concatenation with +.

# Using format() - cleaner name = "John" age = 25 message = "Hello {}, you are {} years old".format(name, age) # Without format() - more writing message = "Hello " + name + ", you are " + str(age) + " years old" # Modern way with f-strings (even better) message = f"Hello {name}, you are {age} years old"
5.
What does the in operation do (between an element and a list)?

Correct Answer:

The in operator checks if an element exists in a list (or other sequence). It returns True if the element is found, False if not.

fruits = ["apple", "banana", "cherry"] print("apple" in fruits) # True print("orange" in fruits) # False # Can also be used with strings text = "Hello World" print("World" in text) # True print("xyz" in text) # False
6.
Create a string containing your full name: first + last name (separated by space). Then:
  • Print the last five characters of the string
  • Print the first third of the string
  • Count how many times the letter 'a' appears in the string
  • Write a command that checks if the letter 'b' appears in the string (returns true/false)
  • Split the string into a list with the first cell containing your first name and the second cell containing your last name
  • Reverse the order of elements in the list
  • Change the last name to uppercase letters
  • Remove the middle character from your first name
  • Create and print a string containing your full name after the changes

Correct Answer:

# Create full name string full_name = "John Smith" # Replace with your actual name print(f"Full name: {full_name}") # Print last five characters print(f"Last 5 characters: {full_name[-5:]}") # Print first third of the string first_third_length = len(full_name) // 3 print(f"First third: {full_name[:first_third_length]}") # Count how many times 'a' appears a_count = full_name.count('a') print(f"Letter 'a' appears {a_count} times") # Check if 'b' appears in the string has_b = 'b' in full_name print(f"Contains letter 'b': {has_b}") # Split into list [first_name, last_name] name_list = full_name.split() print(f"Name list: {name_list}") # Reverse the order of elements name_list.reverse() print(f"Reversed list: {name_list}") # Change last name (now first in list) to uppercase name_list[0] = name_list[0].upper() print(f"After uppercase: {name_list}") # Remove middle character from first name (now second in list) first_name = name_list[1] middle_index = len(first_name) // 2 modified_first = first_name[:middle_index] + first_name[middle_index+1:] name_list[1] = modified_first # Create and print final string final_name = f"{name_list[1]} {name_list[0]}" print(f"Final name: {final_name}")

Print the position of the first + last letter 'o' in the sentence: "Hello world!"
  • Print the sentence from the beginning up to the first 'o'
  • Print the sentence from the last 'o' to the end

9.
How do you write conditions in Python? What is the difference between else and elif?

Correct Answer:

Conditions in Python use if, elif, and else:

else: Executes when the if condition is False (only one else per if)

elif: "else if" - allows multiple conditions to be checked in sequence

age = 18 if age < 13: print("Child") elif age < 18: print("Teenager") elif age < 65: print("Adult") else: print("Senior") # Simple if-else if age >= 18: print("Can vote") else: print("Cannot vote")
10.
How does Python know if lines of code belong to an if statement or not?

Correct Answer:

Python uses indentation (spaces or tabs) to determine which lines belong to an if statement. All lines with the same indentation level after the colon (:) belong to that block.

if age >= 18: print("You are an adult") # This line is indented - belongs to if print("You can vote") # This line is indented - belongs to if print("This runs regardless") # No indentation - outside if block # Wrong indentation causes IndentationError if age >= 18: print("This will cause an error") # Missing indentation!
