üìö Theory Questions

1.
What is a function?
A function is a reusable block of code that performs a specific task. It can accept input (parameters), process that input, and return a result.
2.
How do you define a function in Python?
Use the def keyword followed by the function name, parentheses, and a colon. For example: def my_function():
3.
How do you add parameters to a function? What is the purpose of parameters? How can you give default values to parameters in Python? What is the rule regarding default value assignment (in terms of alignment)?
Parameters are added inside the parentheses. They allow functions to accept input. Default values are assigned using =. Rule: Parameters with default values must come after parameters without defaults.
4.
How can you pass a value to a specific parameter in a Python function?
Use keyword arguments: function_name(parameter_name=value)
5.
Why is it beneficial for functions to return values? How does a Python function return a value?
Returning values allows functions to provide results that can be used elsewhere. Use the return statement.
6.
What does None mean in Python? When does it appear in functions?
None represents the absence of a value. Functions return None by default if no return statement is specified.
7.
How do you generate a random number in Python? Which library do you use? How do you import the library into your code?
Use the random library. Import with import random. Generate numbers with random.randint() or random.random().
